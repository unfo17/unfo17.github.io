Clash

发表于 2023-05-08 更新于 2023-05-11 分类于 software

--Pang

Record:

1、安装

从 Github项目-Clash 中获取安装包并安装，安装流程不作介绍

  • Clash for Windows
  • Clash for Android

2、配置

本文中提到的“订阅”，“节点”等关键词的实质不会进行提供也不会提供获取方法，请自行获取！

For Windows

1、点击左侧“Profiles””
2、“链接与yml文件”订阅



2.1、将订阅链接粘贴进框内
2.2、点击”Download”按钮开始下载配置
2.3、点击上方“Import”，选中“yml文件”后点击“打开”按钮






3、点击左侧“General”
4、点击“System Proxy”开启系统代理
5、点击“Start with Windows”开启开机自启动(可选)



6、点击左侧“Proxies”
7、点击上方“Rule”
8、选择“示例：节点1”即可


  • 其他功能请自行探索

    • For Android

    1、点击“配置“按钮




    2、点击右上角”+“按钮





    3、“链接与yml文件”订阅
    3.1、点击”文件“按钮，选择正确的”yml文件“
    3.2、点击”URL按钮“





    4、将”订阅链接“粘贴进”URL框”内
    5、点击保存
    按需更改其他配置，如名称、自动更新等……
